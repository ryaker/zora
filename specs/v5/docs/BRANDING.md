# Zora v0.5 - Visual Style (Discovery-Future LCARS)

This guide captures the visual language from the Zora header image and defines reusable UI motifs, palette, and typography for consistent repo presentation.

## Core Aesthetic

- Deep, space-black base with subtle scanline texture
- Glassy panels with soft inner glow
- Thin, precise rails and segmented blocks (LCARS-inspired)
- Minimal, confident layout with a few strong anchors

## Palette (extracted/approximated)

Primary:
- Deep black: #0A0B0F
- Midnight navy: #0E1622
- Cyan glow: #7FE7FF
- Teal: #24C7C0
- Amber accent: #F2A84A

Neutrals:
- Cool gray: #5E6B78
- Muted slate: #2A3342
- Ice text: #DDF6FF

Suggested gradients:
- Panel glow: #24C7C0 -> #7FE7FF
- Ambient background: #0A0B0F -> #0E1622

## Typography

Headings:
- Preferred: Eurostile or Bank Gothic
- Web-safe alternatives: Orbitron, Rajdhani

Body:
- Clean sans: Inter or Source Sans 3

System labels:
- Mono: Space Mono or IBM Plex Mono

## Layout Motifs

- Left rail with stacked segments (status modules)
- Top rail band with small segmented blocks
- Central glass panel for title
- Right-side orbital data cluster (circular nodes)

## Usage Guidelines

- Maintain generous negative space
- Use amber sparingly for critical highlights
- Use cyan/teal for primary interactive elements
- Avoid heavy gradients outside glass panels

## Recommended Banner Sizes

- README header: 1600x420 (or 1920x500)
- Safe area for title: center 60% width
- Maintain 40px padding from edges
